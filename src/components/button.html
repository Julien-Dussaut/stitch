Button

The Button component is a fundamental UI element that allows users to perform actions or navigate through the interface. It is designed to be versatile and can be used in various contexts, such as submitting forms, triggering events, or linking to other pages. The Button component supports different styles, sizes, and states to accommodate a wide range of use cases and enhance the user experience.

How it works

The button is built on a single base class st-btn with variant modifiers following BEM convention. All visual properties consume tokens - no hardcoded values in the component itself.

st-btn

Base class for the button component, providing core styles and structure.

--primary

Applies the primary color scheme to the button, making it stand out as the main action.

--outline

Gives the button an outlined style, with a transparent background and a border.

--ghost

Creates a ghost button with no background or border, relying on text color for visibility.
