# Stitch 🪡

> **[Lire en français](#français)**

A lightweight design system and component library built for developers who care about accessibility and theming.

## ✨ Features

- 🎨 **Multiple themes** — Switch between Stich dark, Stitch light, Ember and Midnight
- ♿ **Accessibility first** — WCAG contrast scores built-in
- 🧩 **Ready-to-use components** — Buttons, badges, cards and more
- ⚡ **CLI support** — *(coming soon)* Initialize and add components in seconds

## 🚀 Getting started

```bash
# 1. Initialize
npx stitch init

# 2. Pick a theme
npx stitch theme midnight

# 3. Add components
npx stitch add button card modal
```

> ⚠️ CLI is not yet available. Coming soon.
